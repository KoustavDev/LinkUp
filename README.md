## **Getting Started**  

1. Clone the repository:  
   ```bash
   git clone https://github.com/your-username/linkup.git
   cd linkup
   ```  
2. Install dependencies:  
   ```bash
   npm install
   ```  
3. Set up your **`.env`** file following **`.env.example`**.  
4. Start the development server:  
   ```bash
   npm run dev
   ```
